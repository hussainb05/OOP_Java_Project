# Bug Detective AI 🔍

A Java-based code analyzer that helps beginners detect common programming mistakes in their Java code.

---

## 📋 Project Description

Bug Detective AI allows users to paste Java code into the program, which then scans it line by line to identify common errors such as:
- Missing semicolons
- Incorrect use of `=` instead of `==` in conditions
- Empty `if` blocks
- Possible division-by-zero errors

After analyzing the code, it generates a detailed bug report showing all detected issues along with their corresponding line numbers.

## 🎯 Purpose

Many beginner programmers struggle to identify simple but common coding mistakes. Bug Detective AI solves this by providing an automated, easy-to-use tool that scans Java code and pinpoints errors instantly — helping students learn faster and write cleaner code.

---

## 🗂️ Main Modules & Classes

| Module / Class | Description |
|---|---|
| `BugDetective` | Main entry point; handles user input and coordinates the analysis |
| Bug Detection Logic | Scans code line by line for common Java errors |
| Report Generator | Compiles and displays the bug report with line numbers and issue descriptions |

## ▶️ How to Run

### Requirements
- Java JDK 21 (Java 21.0.9 LTS recommended)

### Steps

1. **Clone the repository:**
   ```
   git clone https://github.com/hussainb05/OOP_Java_Project.git
   ```

2. **Navigate to the project folder:**
   ```
   cd OOP_Java_Project
   ```

3. **Compile the program:**
   ```
   javac BugDetective.java
   ```

4. **Run the program:**
   ```
   java BugDetective
   ```

5. Paste your Java code into the program when prompted and press Enter to receive your bug report.
